Implement a Web application which users can create tasks and assign to another one.

Application Features:

  • User can have one of two possible roles: Admin, Support
  • User needs to login to the system.
  • Support User can create a task.
  • Support User can delete a task if he is the owner.
  • Support User can assign a task to another one.
  • Support User can see the list of all tasks which assigned to him, and change the status to Done.
  • Admin User can see the list of all tasks with Delete permission.
  • Account management like Create User, Change password and etc. is not required.
  • Insert the predefined data to database using Seed of EF

Required technologies in your assignment:

  • - Asp.net (.Net core is recommended)
  • - SQL Server
  • - Entity Framework Code first
  • - Front-End with Angular 2+

Bonus points for: (which are not required, but help us for a better evaluation)

  • • Repository Pattern
  • • Unit of Work
  • • n tier Architecture or DDD
  • • Data Concurrency
  • • Test Module
  • • Logging and Exception Handling
  • • Anything else you think worse to add
